⚙️ Cloud-Sync einrichten

Mit JSONBin.io könnt ihr beide auf dieselben Daten zugreifen – kostenlos, kein Server nötig. Jeder trägt den gleichen API-Key und die gleiche Bin-ID ein.

📋 Einmalige Einrichtung (nur eine Person nötig):
1. Konto auf jsonbin.io erstellen
2. Nach Login: oben rechts → API Keys → Key kopieren
3. Links → Bins → + Create Bin → Inhalt: {} → speichern
4. Bin-ID aus der URL kopieren (z.B. 64abc1234...)
5. Beides hier eintragen und speichern → Link an Partner schicken, dieser trägt dieselben Daten ein
Wird nur lokal in eurem Browser gespeichert
